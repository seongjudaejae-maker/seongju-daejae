# 연등(초) 발원 게시판 — 설정 안내

이 게시판은 방문자가 비밀번호만으로 글을 쓰고, 그 비밀번호로 직접 수정·삭제할 수 있는 게시판입니다.
동시에 관리자는 별도의 "관리자 비밀번호(마스터키)"로 모든 글을 조회·삭제할 수 있습니다.

## 1. 꼭 필요한 설정: 관리자 비밀번호(마스터키) 등록

Netlify 사이트 대시보드에서 아래 경로로 들어가 환경변수를 등록해 주세요.

```
Site settings → Environment variables → Add a variable
```

- **Key**: `ADMIN_MASTER_KEY`
- **Value**: 원하시는 관리자 비밀번호 (예: 영문+숫자 조합 12자 이상 권장)

이 값을 등록하지 않으면, 관리자 화면(`/admin/yeondeung.html`)에서 어떤 비밀번호를 입력해도 인증이 통과되지 않습니다(보안을 위해 일부러 그렇게 만들어 두었습니다).

환경변수를 새로 등록하거나 변경한 뒤에는 **사이트를 한 번 다시 배포(Trigger deploy)** 해야 함수에 반영됩니다.

## 2. 작동 구조 요약

- 방문자가 "연등(초) 발원" 페이지에서 글을 쓰면, 직접 정한 비밀번호와 함께 글이 저장됩니다. (비밀번호는 암호화된 형태로만 저장되어, 관리자도 원문을 볼 수 없습니다.)
- 글 수정·삭제 시 작성자는 그 비밀번호를 입력해야 합니다.
- 관리자는 `/admin/yeondeung.html` 에서 마스터키를 입력하면, 비밀번호 없이도 모든 글을 보고 삭제할 수 있습니다.
- 데이터는 Netlify Blobs(사이트에 내장된 저장 공간)에 보관되며, 별도 데이터베이스를 만들지 않아도 됩니다.

## 3. 관리자 화면 접속 경로

```
https://[본인 도메인]/admin/yeondeung.html
```

`/admin/` 메인 화면(Decap CMS) 우측 하단에도 이 페이지로 가는 링크가 있습니다.

## 4. 주의 사항

- 관리자 비밀번호(마스터키)는 외부에 노출되지 않도록 주의해 주세요. 이 값을 아는 사람은 모든 발원 글을 삭제할 수 있습니다.
- 작성자 비밀번호를 잊어버린 방문자는 본인 글을 직접 수정·삭제할 수 없습니다. 이 경우 관리자에게 문의하도록 안내해 주세요.
- 이 게시판은 신청서 형태이므로, 스팸 방지를 위해 추후 자동입력 방지(캡차) 등을 추가하고 싶으시면 말씀해 주세요.
